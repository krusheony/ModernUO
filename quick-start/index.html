    Quick Start¶

    If you are familiar with RunUO, then follow these steps to get your ModernUO server running in less than 10 minutes:

    1. Download and install the .NET 5 Runtime
    2. Download and extract ModernUO to a folder.
    3. Navigate to the Distribution folder.
    4. Run ModernUO.exe
    1. Download and install the .NET 5 Runtime
    2. Download and extract ModernUO to a folder.
    3. Using terminal, navigate to the Distribution folder.
    4. Run ./dotnet ModernUO.dll
    1. Download and install the .NET 5 Runtime
    2. Download and extract ModernUO to a folder.
    3. Using bash, navigate to the Distribution folder.
    4. Run ./dotnet ModernUO.dll

    Note

    Follow the rest of the Get Started guide to learn how to connect, configure, and customize, your server.
